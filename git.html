TABLEAU DES COMMANDES GIT

git config Définir une préférence utilisateur
git init Créer un dépôt avec Git
git add Ajouter un fichier à l'index
git clone Copier un dépôt local ou distant
git commit Valider une modification du head
git status Affiche la liste des fichiers modifiés ou en cours de validation
git push Envoyer une modification vers la branche principale
git checkout Créer une branche, ou naviguer entre celle qui existent déjà
git remote Se connecter à un dépôt distant et vérifier ceux auxquelles nous somme déjà connecté
git branch Répertorier, créer ou supprimer des branches
git pull Fusionner toutes les modifications présentes sur le dépôt distant, et les ajouter au répertoire local
git merge Fusionner une branche avec la branche active
git diff Dresse une liste des conflits
git tag Se situer
git reset Réinitialiser l'index et le répertoire de travail à l'état du dernier commit
git rm Supprimer des fichier de l'index et du répertoire de travail
git stash Enregistrer les changements qui ne doivent pas être commit directement
git show Afficher les informations d'un fichier git
git fetch Extraire tous les fichiers d'un dépôt distant qui ne sont pas dans le répertoire de travail
git ls-tree Afficher un fichier arborescent avec le nom et le mode de chaque élément
git cat-file Afficher le type d'un fichier
git grep Rechercher un élément dans le répertoire
gitk Interface graphique du dépôt local
git instaweb Exécuter un serveur web par interface
git gc Optimiser le dépôt en supprimant les fichiers inutiles
git archive Compresser les composantes d'un arbre du dépôt en ZIP ou TAR
git prune Les fichiers qui n'ont pas de pointeurs entrant seront supprimés
git fsck Identifier les fichiers corrompus
git rebase Réappliquer un commit sur une autre branche
